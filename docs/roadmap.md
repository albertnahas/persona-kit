# Roadmap

## Current (v0.1)

- [x] Agent creation with persona and instructions
- [x] RAG with markdown document loader
- [x] In-memory vector store
- [x] Memory persistence (KV, SQLite)
- [x] Streaming responses via Vercel AI SDK
- [x] React hook (`useChatAgent`)
- [x] Next.js template

## Near-term

### v0.2 - Enhanced RAG

- [ ] PDF document loader
- [ ] Web crawler loader
- [ ] Chunking strategies (semantic, recursive)
- [ ] Hybrid search (keyword + vector)
- [ ] Metadata filtering in search

### v0.3 - More Providers

- [ ] Anthropic Claude support
- [ ] Cohere embeddings
- [ ] Local embeddings (Ollama)
- [ ] Fireworks AI

### v0.4 - Better DX

- [ ] CLI tool for scaffolding
- [ ] Hot reload for persona/instructions
- [ ] Debugging tools
- [ ] Usage analytics
